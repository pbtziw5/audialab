## The Products

### Emergent Drums 2 — The Main Event

This is their flagship, and it does something a regular sample library physically cannot do: generate drum samples that have never existed before, on demand, offline, in your DAW.

The interface is a 16-pad layout. Each pad generates a different hit type — kick, snare, hat, whatever. Hit generate, get a new sound. Hate it? Hit generate again. You can do this forever. There's a similarity slider so you can say "I like this kick, give me something in the same ballpark" without starting from scratch each time.

Two generation modes: **Creamy** (polished, clean sounds suited for pop and electronic) and **Crunchy** (grittier, more experimental, great for lo-fi or industrial stuff). Once you have a sound you want, you drag it straight into your DAW. No export dialog, no file management.

It runs locally now, which is a big improvement over the first version. You don't need an internet connection to generate sounds.

### Interloper — The New One

Interloper is a multiband interpolation engine, which sounds intimidating but the concept is simple: load two samples, then blend between their low, mid, and high frequency bands independently.

So if you want the sub weight of Kick A but the snap crack of Kick B, you can literally dial that in with a slider. No EQ wrestling, no phase cancellation nightmares, no building a routing chain from scratch. Interloper just handles it.

It's MIDI-driven and fully automatable, so the blend ratios can change over time. That means sounds can morph in real time as a pattern plays — something you'd previously have to do with a pretty annoying automation chain.

It launched at an intro price and is part of the current sale.

### Humanize — Small Tool, Real Problem

Here's the thing about programmed drums: if you play the exact same sample on every hit, it sounds like a machine gun. Real drummers never hit identically twice. Humanize fixes that.

Drop a sample in, it generates 8–9 variations with subtle differences in pitch, timing, and timbre. Those variations play back in round-robin so consecutive hits are always slightly different from each other. Two modes: **Humanize** (subtle, realistic variations) and **Superhuman** (more extreme, experimental).

It's a narrow tool for a specific problem. If you've never noticed your drum patterns sounding robotic, you don't need it. If you have, you'll want it.

👉 [Get Humanize here](https://audialab.com/products/humanize/?ref=li15)

### Infinite Packs & Deep Sampler 2

Infinite Packs is what it sounds like — AI-generated sample packs across genres, all royalty-free. Less technically impressive than Emergent Drums, but useful if you want to bulk up your library without licensing headaches.

Deep Sampler 2 is **free**. You type a description ("punchy 808 with some room"), it generates a sample. Good for quick reference sounds and prototyping. At zero dollars, there's no real argument against trying it.
